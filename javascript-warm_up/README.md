# Holberton School - JavaScript - Warm up

## Description

This project is a comprehensive study of foundational JavaScript programming concepts, focusing on building solid problem-solving skills and understanding core language mechanics. Over the course of **18 specialized tasks (including advanced sections)**, I implemented a series of scripts from scratch, focusing on syntax accuracy, logical branching, loops, functions, and object manipulation without the use of high-level external libraries.

## Learning Objectives

By completing this project, I am able to explain and implement:

*   **Language Power:** Why JavaScript programming is amazing and its versatility.
*   **Execution:** How to run a JavaScript script using Node.js environments.
*   **Variables & Constants:** How to create variables and constants, and the exact differences between `var`, `const`, and `let`.
*   **Data Types:** All the data types available in JavaScript (Primitives and Objects).
*   **Control Flow:** How to use `if` and `if ... else` conditional statements.
*   **Code Documentation:** How to properly use single-line and multi-line comments.
*   **Value Assignment:** How to assign values to variables using assignment operators.
*   **Iteration:** How to utilize `while` and `for` loops effectively.
*   **Loop Control:** How to apply `break` and `continue` statements to manage loop executions.
*   **Modular Logic:** What a function is, how to use functions, and understanding what a function returns when no explicit `return` statement is defined (`undefined`).
*   **Variable Lifetime:** Understanding the Scope of variables (Global, Function, and Block scope).
*   **Mathematics:** What the arithmetic operators are and how to use them.
*   **Data Structures:** How to manipulate dictionaries (JavaScript Objects).
*   **Code Reusability:** How to import and export functions and files.

## Requirements

To comply with Holberton School standards:

*   **Allowed Editors:** `vi`, `vim`, `emacs`.
*   **Environment:** All files will be interpreted and tested on **Ubuntu 20.04 LTS** using **node (version 14.x)**.
*   **Formatting Rules:** 
    * All files must end with a new line.
    * The first line of all files must be exactly `#!/usr/bin/node` (Shebang).
    * Code must be **semistandard** compliant (version 16.x.x) — Rules of Standard + semicolons on top. AirBnB style is used as a reference.
*   **Permissions:** All files must be executable (`chmod +x`).
*   **Validation:** The length of files will be tested using the `wc` utility.
*   **Mandatory:** A `README.md` file at the root of the project folder.

## How to launch the Project

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/mammadovs/holbertonschool-higher_level_programming.git
    ```
2.  **Navigate to the project folder:**
    ```bash
    cd holbertonschool-higher_level_programming/javascript-warm_up
    ```
3.  **Make files executable:**
    ```bash
    chmod +x *.js
    ```
4.  **Execute the scripts:**
    * Run using the shebang directly:
      ```bash
      ./0-javascript_is_amazing.js
      ```
    * Or run explicitly with Node.js:
      ```bash
      node 0-javascript_is_amazing.js
      ```
    * Pass arguments where required:
      ```bash
      ./3-value_argument.js "Holberton"
      ```

## Project Milestones

The 18 tasks were organized into the following development phases:

1.  **Basic Syntax & Output (Tasks 0-1)**

    Initializing constants and printing messages to the console.
    Working with string literals and simple console layout outputs.

2.  **Arguments & Data Conversion (Tasks 2-5)**

    Parsing process arguments (process.argv) to dynamically change script output based on user input.
    Working with type coercion and string concatenation.
    Safe typecasting using parseInt and handling edge cases with isNaN.

3.  **Loops & Math Logic (Tasks 6-10)**

    Array structural iteration using classic loops (for, while).
    Handling nested structural loops to draw complex characters and matrix configurations (e.g., drawing squares).
    Defining custom computation systems including algebraic formulas and factorial calculations.

4.  **Advanced Array & Object Architectures (Tasks 11-13)**

    Implementing custom sorting and maximum tracking algorithms on variable argument lists.
    Dictionary (Object) management, key-value manipulation, and external system file inclusion (require).

5.  **Advanced Scope & Functional Concepts (Tasks 14-17 / Advanced)**

    Bypassing variable scope structures to modify external context values.
    Building higher-order functions that interact with functional callbacks.
    Embedding methods directly inside objects to internally increment state attributes.
