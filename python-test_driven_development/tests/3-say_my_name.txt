===============================
How to use 3-say_my_name module
===============================

First, import the function:
    >>> say_my_name = __import__('3-say_my_name').say_my_name

Basic usage with two strings:
    >>> say_my_name("John", "Smith")
    My name is John Smith

Usage with only one string (last_name defaults to empty):
    >>> say_my_name("Walter")
    My name is Walter 

Usage with an empty string as last name:
    >>> say_my_name("Bob", "")
    My name is Bob
