===============================
How to use 0-add_integer module
===============================

Importing the function:
    >>> add_integer = __import__('0-add_integer').add_integer

Normal cases:
    >>> add_integer(1, 2)
    3
    >>> add_integer(100, -2)
    98
    >>> add_integer(2)
    100
    >>> add_integer(100.3, -2)
    98
    >>> add_integer(4, 5.5)
    9

Edge cases:
    >>> add_integer(0, 0)
    0
    >>> add_integer(-1, -1)
    -2
