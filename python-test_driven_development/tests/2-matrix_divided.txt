The ``2-matrix_divided`` module
===============================

This module provides a function ``matrix_divided(matrix, div)`` that divides all elements of a matrix by a given number.

Usage:
------

Import the function:
    >>> matrix_divided = __import__('2-matrix_divided').matrix_divided

Normal cases:
    >>> matrix = [[1, 2, 3], [4, 5, 6]]
    >>> matrix_divided(matrix, 3)
    [[0.33, 0.67, 1.0], [1.33, 1.67, 2.0]]

    >>> matrix = [[10.5, 44.0], [1.2, 3.3]]
    >>> matrix_divided(matrix, 2)
    [[5.25, 22.0], [0.6, 1.65]]

    >>> matrix_divided([[1, 2], [3, 4]], 1)
    [[1.0, 2.0], [3.0, 4.0]]

Check if the original matrix remains unchanged (immutability):
    >>> matrix = [[2, 4], [6, 8]]
    >>> new_matrix = matrix_divided(matrix, 2)
    >>> matrix
    [[2, 4], [6, 8]]
